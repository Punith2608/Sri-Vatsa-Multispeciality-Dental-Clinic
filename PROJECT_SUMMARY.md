## 🚀 Quick Start (Follow These Steps)

### Step 1: Install Dependencies
```bash
cd c:\Users\tirup\Desktop\punith\dental-clinic-website
npm install
```

### Step 2: Configure Firebase (Optional but Recommended)
```
Create .env.local file with:
REACT_APP_FIREBASE_API_KEY=your_key
REACT_APP_FIREBASE_PROJECT_ID=your_project_id
(See .env.example for all variables)
```

### Step 3: Run Development Server
```bash
npm start
```
Website opens at `http://localhost:3000`

### Step 4: Customize (Edit your clinic info)
- Update `src/pages/Contact.jsx` with clinic address
- Update `src/components/Navigation.jsx` with phone
- Update doctors in `src/pages/Team.jsx`
- Update services in `src/pages/Services.jsx`

### Step 5: Deploy
```bash
# Option A: Vercel (Easiest)
npm install -g vercel
vercel

# Option B: Netlify
npm run build
netlify deploy --prod --dir=build
```
