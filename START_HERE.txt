🔧 SETUP STEPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Install Dependencies
────────────────────────────────────────────────────────────────────────────
npm install

STEP 2: Configure Firebase (Optional but Recommended)
────────────────────────────────────────────────────────────────────────────
Create .env.local file with:

REACT_APP_FIREBASE_API_KEY=your_key_here
REACT_APP_FIREBASE_AUTH_DOMAIN=your_domain.firebaseapp.com
REACT_APP_FIREBASE_PROJECT_ID=your_project_id
REACT_APP_FIREBASE_STORAGE_BUCKET=your_bucket.appspot.com
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
REACT_APP_FIREBASE_APP_ID=your_app_id

Get these from: https://console.firebase.google.com/

STEP 3: Start Development Server
────────────────────────────────────────────────────────────────────────────
npm start

Opens at: http://localhost:3000

STEP 4: Customize Clinic Information
────────────────────────────────────────────────────────────────────────────
✓ Update src/components/Navigation.jsx - Phone, clinic name
✓ Update src/components/Footer.jsx - Contact info
✓ Update src/pages/Contact.jsx - Address, email, phone
✓ Update src/pages/Team.jsx - Doctor profiles
✓ Update src/pages/Services.jsx - Service details
✓ Update src/pages/Appointment.jsx - Time slots, treatments

STEP 5: Test Locally
────────────────────────────────────────────────────────────────────────────
• Open http://localhost:3000
• Test all pages
• Test forms
• Test on mobile (F12 > Toggle device toolbar)
• Check console for errors

STEP 6: Build for Production
────────────────────────────────────────────────────────────────────────────
npm run build

STEP 7: Deploy
────────────────────────────────────────────────────────────────────────────

Option A: Vercel (Recommended)
┌─────────────────────────────────────────────────────────────────────────┐
│ npm install -g vercel                                                   │
│ vercel                                                                  │
│ Follow prompts → Done!                                                  │
└─────────────────────────────────────────────────────────────────────────┘

Option B: Netlify
┌─────────────────────────────────────────────────────────────────────────┐
│ npm run build                                                           │
│ npm install -g netlify-cli                                             │
│ netlify deploy --prod --dir=build                                      │
└─────────────────────────────────────────────────────────────────────────┘

ISSUE: npm install fails
FIX:
  rm -rf node_modules package-lock.json
  npm install

ISSUE: Styles not loading
FIX:
  npm install
  Ctrl+Shift+Delete (clear cache)
  npm start

ISSUE: Firebase not connecting
FIX:
  ✓ Check .env.local exists
  ✓ Verify credentials match Firebase Console
  ✓ Restart dev server
  ✓ Check browser console for errors

ISSUE: Port 3000 already in use
FIX:
  PORT=3001 npm start

ISSUE: Build fails
FIX:
  rm -rf node_modules package-lock.json build
  npm install
  npm run build
